## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need the followings installed and running on your system:

- Git
- Ruby
- Rails
- PostgreSQL

<!--
Example command:

```sh
 gem install rails
```
 -->

### Setup

Clone this repository to your desired folder:

<pre>git clone git@github.com:kkolade/rails_blog_app.git</pre>
<!--
Example commands:

```sh
  git git clone git@github.com:kkolade/rails_blog_app.git
  cd blog
```
--->

### Install

Install the gems required for this project with:

```sh
  cd my-project
  gem install
```
--->

### Usage

To run the project, execute the following command:

<pre>rails s</pre>

Open your browser and go to:
<pre>http://localhost:3000/</pre>

### Run tests

To run tests, run the following command:

<pre>rspec ./spec/</pre>
